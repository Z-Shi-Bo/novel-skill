# Urban Mystery Example

## 适用题材
- 都市悬疑
- 犯罪调查
- 现实派刑侦

## 推荐工作流
```text
init
→ brief
→ bible
→ characters
→ outline
→ research
→ write
→ review
→ sync
→ diagnose
```

## 示例 brief 要点
- 核心卖点：城市秩序表面平静，旧案不断回潮
- 目标读者：喜欢现实悬疑、社会质感、追索真相
- 风格基调：冷硬、克制、压迫感

## 示例 bible 要点
- 警务、法医、媒体流程不能乱写
- 城市场景必须有可见物和真实生活纹理
- 线索必须可追溯，不靠天降答案

## 示例 outline 要点
- 卷 1：旧案重开
- 卷 2：系统性遮蔽
- 卷 3：真相交换成本

## 示例 review 重点
- 线索公平性
- 节奏控制
- 程序正义 / 现实流程可信度
