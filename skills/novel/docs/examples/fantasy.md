# Fantasy Example

## 适用题材
- 玄幻
- 奇幻
- 长篇升级流

## 推荐工作流
```text
init
→ brief
→ bible
→ characters
→ outline
→ state/resource setup
→ write
→ review
→ sync
→ diagnose
```

## 示例 brief 要点
- 核心卖点：高压规则 + 强回报推进
- 目标读者：偏成长线、爽点、设定闭环
- 风格基调：明确、利落、冲突驱动

## 示例 bible 要点
- 力量体系必须分层清楚
- 资源增减要可追踪
- 规则不能为当前剧情临时变形

## 示例 state 层重点
- `resource_ledger.md`
- `pending_hooks.md`
- `current_state.md`

## 示例 review 重点
- 战力是否飘
- 资源是否乱
- 爽点是否结算
- 设定是否吃书
