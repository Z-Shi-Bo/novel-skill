# Research Policy

## 何时触发

必触发：

- 历史年代
- 地理与机构细节
- 行业 / 职业流程
- 法律 / 政策
- 金融 / 经济 / 物价
- 武器 / 装备 / 交通工具
- 医疗 / 刑侦 / 执法程序
- 社会风俗与时代物件

可选触发：

- 饮食、服饰、街景等低风险氛围细节
- 行话、器物用法等强化信息

不触发：

- 纯架空世界内部规则
- 虚构地图与虚构机构
- 纯情绪描写
- 不依赖真实事实的剧情讨论

## 工作流

```text
发现不确定点
→ 拆成具体问题
→ 搜索与交叉验证
→ 生成 research note
→ 提炼为可写事实
→ 再进入写作或审稿
```

## 落盘要求

research note 路径：
`06_reports/research/{topic-slug}.md`

note 至少包含：

- 研究问题
- 适用章节 / 场景
- 核验结论
- 可直接用于创作的事实摘录
- 冲突点
- 来源与日期

## 原则

1. research note 不直接等于正文。
2. 外部事实不覆盖小说内部既成剧情事实。
3. 查不到可靠答案时，给保守写法，不装作查过。
4. 研究结果先变成 note，再变成可写事实。
