# Period Fiction Example

## 适用题材
- 历史年代
- 民国 / 改革开放 / 旧上海 / 旧港澳

## 推荐工作流
```text
init
→ brief
→ bible
→ timeline
→ research
→ outline
→ write
→ review
→ sync
→ diagnose
```

## 示例 brief 要点
- 核心卖点：时代洪流中的个人命运与利益链
- 目标读者：喜欢年代质感、制度摩擦、社会变局
- 风格基调：写实、克制、重细节

## 示例 bible 要点
- 时间线、制度、称呼、货币与器物必须稳
- 不要把现代语言和现代心理硬塞进旧时代文本

## 示例 research 重点
- 机构名称
- 职业流程
- 常见物件
- 货币与价格
- 风俗与称谓

## 示例 review 重点
- 年代错配
- 风俗错位
- 时代感是否落到可见细节
